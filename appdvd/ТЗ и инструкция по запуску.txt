Техническое задание:

Необходимо написать автоматизацию ресурса по обмену dvd дисками, веб-приложение. Есть коллекция дисков у каждого участника. Диски можно брать и отдавать. В системе три сущности: Disk (DVD-диск), User, TakenItem (связка User-Disk)

В системе должно быть пять страниц: авторизация, заведение нового диска, список свободных дисков (у всех пользователей невзятые), список дисков, взятых пользователем и список дисков,  взятых у пользователя с указанием кто взял. Диск можно взять и отдать (без денежных расчетов).

СУБД – hsqldb. Нужная схема должна автоматически создаваться при первом запуске разработанной программы. Начальные данные также должны автоматически заводиться.

Работа с СУБД - Hibernate. Обязательно использование HQL или Criteria Builder.

Проект должен быть написан с использованием MVC-фреймворка Spring. Авторизация в системе – с использованием Spring Security.

Представление - JSP. Для отображения списков можно использовать стандартный JSTL, либо любую другую библиотеку тегов.

Сервер приложений - встроенный Jetty, не требующий внешнего веб-контейнера.

Проект должен автоматически собираться и запускать на компьютере с предустановленными JDK и Maven.


Инструкция:

1. 2 раза кликнуть по файлу "run.bat" (только на win, должны быть установлены JDK и Maven, Maven должен быть прописан в path ОС)
2. Перейти по адресу http://127.0.0.1:8080/dvd/auth
